## 五、使用说明

### 在游戏中使用

```javascript
// 播放 BGM
const bgm = new Audio('/assets/audio/bgm/menu_theme.mp3');
bgm.loop = true;
bgm.play();

// 播放音效
const sfx = new Audio('/assets/audio/sfx/ui_click.ogg');
sfx.play();
```

### 署名要求

由于 BGM 使用 CC BY 4.0 许可证，请在游戏制作人员名单或关于页面中添加：

```
Music by Tanner Helland
https://github.com/tannerhelland/free-music
```
