# ⚔️ 蜀山剑侠传

> 蜀山万载，剑气纵横三万里

一款基于还珠楼主经典小说《蜀山剑侠传》改编的文字修仙游戏。

## 🎮 游戏特色

### 核心玩法
- **修炼养成** - 从练气到渡劫，体验完整的修真之路
- **法宝收集** - 收集数百种法宝，打造专属组合
- **剑光斗法** - 回合策略战斗，五行相克
- **世界探索** - 蜀山世界等你探索

### 游戏亮点
- 🎴 还原原著世界观
- ⚔️ 策略性战斗系统
- 🔮 五行相克系统
- 📜 丰富的法宝体系
- 🌍 开放世界探索

## 🚀 快速开始

### 环境要求
- Python 3.10 或更高版本

### 安装运行

```bash
# 克隆项目
cd ~/.openclaw/workspace/shushan-game

# 运行游戏
python main.py
```

### 游戏操作

```
【主菜单】
1. 开始新游戏 - 创建角色，开启修仙之旅
2. 继续游戏 - 读取存档
3. 游戏说明 - 查看游戏帮助
4. 退出 - 退出游戏
```

## 📖 游戏指南

### 修炼境界

| 境界 | 修为要求 | 特点 |
|------|---------|------|
| 练气期 | 0 | 初入修真 |
| 筑基期 | 1,000 | 可用飞剑 |
| 金丹期 | 10,000 | 剑光分化 |
| 元婴期 | 100,000 | 神识外放 |
| 化神期 | 1,000,000 | 法宝炼化 |
| 合体期 | 10,000,000 | 准备飞升 |
| 渡劫期 | 100,000,000 | 渡劫飞升 |

### 五行相克

```
金 克 木 克 土 克 水 克 火 克 金
```

**克制效果：** 伤害 × 1.5

### 法宝类型

| 类型 | 用途 | 代表法宝 |
|------|------|---------|
| 飞剑 | 主要攻击 | 紫郢剑、青索剑 |
| 防御 | 护身防御 | 五云锁仙屏 |
| 攻击 | 大范围伤害 | 白眉针 |
| 辅助 | 功能性 | 遁光梭 |
| 特殊 | 独特效果 | 天蝉灵叶 |

## 🎯 游戏流程

### 新手指南

1. **创建角色**
   - 输入名字
   - 选择门派（峨眉、青城、昆仑等）

2. **开始修炼**
   - 打坐修炼获得修为
   - 达到阈值后突破境界

3. **收集法宝**
   - 战斗掉落
   - 完成任务
   - 探索世界

4. **提升实力**
   - 装备更强的法宝
   - 学习新功法
   - 挑战更强敌人

### 修炼技巧

- **打坐** - 日常挂机，无消耗
- **吐纳** - 快速获得修为，消耗真元
- **悟道** - 最快修炼方式，消耗较多真元

### 战斗策略

- 观察敌人五行，选择克制法宝
- 合理使用防御，减少伤害
- 高境界压制低境界
- 法宝组合产生特效
