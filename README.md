## 🎮 游戏简介

《蜀山剑侠传》是一款基于还珠楼主同名小说改编的文字修仙游戏。玩家将扮演一名初入修真界的弟子，通过修炼、探索、战斗，逐步成长为一代仙侠。

### 核心玩法
- **修炼养成** - 提升境界、学习功法、收集法宝
- **世界探索** - 探索蜀山世界、触发奇遇事件
- **策略战斗** - 法宝对决、五行克制、阵法系统
- **社交互动** - 好友系统、排行榜、门派战

## 🚀 快速开始

### 在线游玩
**Web 版**: https://shushan-game.vercel.app

### 本地运行

```bash
# 克隆项目
git clone https://github.com/tancau/shushan-game.git
cd shushan-game

# 安装依赖
pip install flask flask-cors

# 运行游戏
python3 main.py
```

### Web API

```bash
# 启动 API 服务
python3 api.py

# 访问
http://localhost:8000
```

## ✨ 功能特性

### 📊 系统概览

| 类别 | 系统 | 数量 | 状态 |
|------|------|------|------|
| **核心系统** | 修炼、法宝、功法、战斗、地图、任务、炼丹炼器 | 7 | ✅ |
| **高级系统** | 阵法、NPC、商店、奇遇、灵兽、天劫、道心、成就 | 8 | ✅ |
| **社交功能** | 排行榜、副本、活动、好友 | 4 | ✅ |

### 🎯 核心系统

#### 1. 修炼系统
- **7 大境界**: 练气期 → 筑基期 → 金丹期 → 元婴期 → 化神期 → 合体期 → 渡劫期
- **多种修炼方式**: 打坐、吐纳、悟道、吞丹
- **功法加成**: 25 种功法，主修/辅修/秘术

#### 2. 法宝系统
- **15 种法宝**: 青竹剑、紫郢剑、轩辕剑...
- **五行相克**: 金克木、木克土、土克水、水克火、火克金
- **法宝强化**: 最高 +10

#### 3. 功法系统
- **25 种功法**: 峨眉剑法、太清一气诀、昆仑剑经...
- **功法类型**: 主修、辅修、秘术
- **功法升级**: 最高 10 级

#### 4. 战斗系统
- **回合策略**: 技能释放、真元消耗
- **伤害计算**: 攻击、防御、五行克制
- **灵兽出战**: 灵兽协助战斗

#### 5. 地图探索
- **33 个地点**: 中原、南疆、北海、西域、东海
- **探索事件**: 6 种事件类型
- **移动系统**: 地点间移动

#### 6. 任务系统
- **21 个任务**: 主线、支线、日常、奇遇
- **目标追踪**: 自动记录进度
- **奖励发放**: 修为、灵石、法宝、功法

#### 7. 炼丹炼器
- **31 种材料**: 矿石、药材、妖丹、灵材
- **27 种配方**: 丹药 14 种、法宝 13 种
- **制作成功率**: 受功法影响

### 🌟 高级系统

#### 8. 阵法系统
- **15 种阵法**: 五行阵、剑阵、困敌阵、辅助阵
- **破阵机制**: 护盾、反噬伤害
- **状态效果**: 减速、冰封、麻痹、混乱

#### 9. NPC 系统
- **25 个 NPC**: 商人、师傅、任务发布者、特殊 NPC
- **对话系统**: 条件对话、选项对话
- **商店交易**: 9 个商店

#### 10. 奇遇系统
- **30 种奇遇**: 传承、宝物、机缘、危险、奇人
- **触发机制**: 多维度条件
- **选择系统**: 不同选择影响结果

#### 11. 灵兽系统
- **25 种灵兽**: 神兽 5、灵兽 13、妖兽 7
- **捕捉培养**: 驯服、喂养、进化
- **战斗辅助**: 出战、技能释放

#### 12. 天劫系统
- **15 种天劫**: 小天劫、大天劫、灭世天劫
- **渡劫机制**: 回合制战斗
- **飞升系统**: 飞升成仙

#### 13. 道心系统
- **功德业力**: 行为影响功德/业力
- **正邪选择**: 正道、魔道、邪道、混沌
- **功德商店**: 8 种兑换物品

#### 14. 成就系统
- **50 个成就**: 修炼、战斗、探索、收集、社交、特殊
- **35 个称号**: 带属性加成
- **成就点数**: 兑换奖励

### 🎊 社交功能

#### 15. 排行榜系统
- **5 大榜单**: 战力榜、修为榜、境界榜、财富榜、成就榜
- **排名奖励**: 每日/每周奖励
- **专属称号**: 前三名称号

#### 16. 副本系统
- **12 个副本**: 普通、精英、传说
- **多层挑战**: 小怪层 + Boss 层
- **副本奖励**: 首通奖励、通关奖励

#### 17. 活动系统
- **12 个活动**: 日常、周常、节日、特殊
- **签到系统**: 每日签到、连续签到奖励
- **积分兑换**: 活动积分兑换奖励

#### 18. 好友系统
- **好友列表**: 添加、删除好友
- **好感度**: 6 个等级
- **互动功能**: 赠送、切磋、聊天
