# 炼妖系统设计文档

> 蜀山剑侠传 - 灵兽捕捉与炼化系统

---

## 一、系统概述

炼妖系统是蜀山剑侠传的核心特色玩法，类似《轩辕剑》系列的炼妖壶系统。

**核心功能**：
- 捕捉灵兽
- 炼化灵兽获得法宝/丹药/技能
- 培养灵兽协助战斗

---

## 二、灵兽捕捉

### 2.1 捕捉条件

| 条件 | 说明 |
|------|------|
| **灵兽HP** | 需降到20%以下 |
| **捕捉道具** | 需要捕捉符/灵兽袋 |
| **捕捉成功率** | 基础30% + (灵兽HP越低越高) |

### 2.2 捕捉道具

| 道具 | 成功率加成 | 价格 |
|------|-----------|------|
| **普通捕捉符** | +10% | 100灵石 |
| **高级捕捉符** | +20% | 500灵石 |

## 三、灵兽分类

### 3.1 五行灵兽

| 五行 | 代表灵兽 | 特点 |
|------|----------|------|
| **金** | 金翅大鹏、金刚猿 | 攻击强 |
| **木** | 木灵、树妖 | 回复强 |
| **水** | 水龙、玄龟 | 防御强 |
| **火** | 火凤、烈焰虎 | 暴击强 |
| **土** | 土行孙、石巨人 | 血量高 |
