## 一、境界系统

### 1.1 七大境界

| 境界 | 修为需求 | 属性加成 | 解锁内容 |
|------|----------|----------|----------|
| **练气期** | 0 | 基础 | 基础技能 |
| **筑基期** | 10,000 | +20% | 中级功法、新技能栏 |
| **金丹期** | 50,000 | +40% | 高级功法、法宝强化 |
| **元婴期** | 200,000 | +60% | 顶级功法、元神技能 |
| **化神期** | 500,000 | +80% | 领域技能 |
| **合体期** | 1,000,000 | +100% | 神通技能 |
| **渡劫期** | 2,000,000 | +150% | 天劫挑战 |

### 1.2 境界突破

```
修为达到要求 → 触发突破任务 → Boss战 → 成功突破
```

## 二、法宝系统

### 2.1 法宝分类

| 类型 | 特点 | 技能类型 |
|------|------|----------|
| **剑类** | 攻击高 | 攻击技能 |
| **印类** | 效果强 | 控制技能 |
| **镜类** | 范围大 | AOE技能 |
| **符类** | 效果多 | 辅助技能 |

### 2.2 法宝品质

| 品质 | 颜色 | 属性倍率 |
|------|------|----------|
| **凡品** | 白色 | 100% |
| **灵品** | 绿色 | 130% |
| **仙品** | 蓝色 | 170% |
| **神品** | 紫色 | 220% |
| **圣品** | 橙色 | 300% |

## 三、功法系统

### 3.1 功法分类

| 类型 | 效果 |
|------|------|
| **主动功法** | 战斗中使用，消耗MP |
| **被动功法** | 自动生效 |
| **修炼功法** | 提升修为获取速度 |

### 3.2 功法学习

```
获得功法书 → 消耗修为学习 → 装备功法 → 生效
```
