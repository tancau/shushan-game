# 地图系统设计文档

> 蜀山剑侠传 - JRPG地图探索系统

---

## 一、地图概述

### 1.1 地图类型

采用 **2D横板地图**，类似《轩辕剑3》风格：
- 多层背景（远景、中景、近景）
- 装饰性建筑和植被
- 可交互元素（NPC、宝箱、机关）

### 1.2 遇敌方式

**踩地雷式**：在地图上走动时随机触发战斗

| 区域类型 | 遇敌率 | 说明 |
|----------|--------|------|
| 安全区 | 0% | 城镇、门派内部 |
| 低危区 | 5%/步 | 新手区、主路 |
| 中危区 | 10%/步 | 野外、洞府外围 |
| 高危区 | 15%/步 | 秘境、妖魔巢穴 |

## 三、地图交互

| 元素 | 交互方式 | 奖励 |
|------|----------|------|
| **NPC** | E键对话 | 任务、信息、交易 |
| **宝箱** | E键开启 | 金钱、物品、法宝 |
| **采集点** | E键采集 | 材料、灵草 |
| **传送点** | E键传送 | 快速移动 |
