## 四、Boss设计

### 4.1 Boss特点

- 多阶段战斗
- 特殊机制
- 稀有掉落

### 4.2 Boss列表

| Boss | 位置 | HP | 阶段 | 弱点 | 掉落 |
|------|------|-----|------|------|------|
| 妖狼王 | 蜀山脚下 | 2000 | 2 | 火 | 狼王牙、新手剑 |
| 剑灵守护 | 蜀山主峰 | 5000 | 3 | 水 | 剑灵晶、中级剑 |
| 峨眉掌门 | 峨眉山 | 10000 | 3 | 木 | 峨眉剑谱、峨眉剑 |

### 4.3 Boss战斗机制

**妖狼王**：
- 阶段1 (100%-50%): 普通攻击 + 召唤狼
- 阶段2 (50%-0%): 狂暴 + 连击

**剑灵守护**：
- 阶段1 (100%-70%): 剑气攻击
- 阶段2 (70%-30%): 剑阵 + 护盾
- 阶段3 (30%-0%): 狂暴剑气

## 五、敌人AI

### 5.1 AI决策逻辑

```
1. 检查HP状态
   - HP < 30% → 使用回复技能(如果有)
   
2. 检查玩家状态
   - 玩家HP低 → 使用攻击技能
   - 玩家有增益 → 使用驱散技能(如果有)
   
3. 随机选择技能
   - 根据技能权重随机选择
   
4. 执行行动
```

### 5.2 AI类型

| 类型 | 行为 |
|------|------|
| **攻击型** | 优先使用攻击技能 |
| **防御型** | 频繁使用防御 |
| **平衡型** | 随机选择行动 |
| **智能型** | 根据战场情况决策 |

## 六、掉落系统

### 6.1 掉落规则

```
每个敌人有掉落列表
每个物品有掉落概率(0-100%)
战斗胜利后随机判定
```

### 6.2 掉落物品类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **材料** | 炼化用材料 | 狼牙、铁矿石 |
| **装备** | 武器防具 | 铁剑、布甲 |
| **消耗品** | 丹药符箓 | 回血丹、传送符 |
| **金钱** | 游戏货币 | 灵石 |
