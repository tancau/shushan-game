### 一、修炼系统

#### 1.1 境界体系

```python
REALMS = {
    "练气期": {
        "level": 1,
        "cultivation_required": 0,
        "description": "感应天地灵气",
        "unlock_features": ["基础法术"]
    },
    "筑基期": {
        "level": 2,
        "cultivation_required": 1000,
        "description": "铸就道基",
        "unlock_features": ["飞剑", "二级法宝"]
    },
    # ... 更多境界
}
```

**设计要点：**
- 境界提升需要修为达到阈值
- 每个大境界解锁新功能
- 境界提升后属性大幅增长
- 高境界对低境界有压制效果

#### 1.2 修炼方式

| 方式 | 修为获得 | 消耗 | 适用场景 |
|------|---------|------|---------|
| 打坐 | 10 | 无 | 日常挂机 |
| 吐纳 | 20 | 5真元 | 有真元时 |
| 悟道 | 50 | 20真元 | 快速提升 |
| 吞丹 | 变化 | 丹药 | 有资源时 |
| 灵脉 | 100 | 占领灵脉 | 资源争夺 |

#### 1.3 功法系统

**功法类型：**
- **主修功法** - 决定修炼速度和核心能力
- **辅修功法** - 炼丹、炼器、阵法等辅助技能
- **秘术** - 特殊技能，如遁术、搜魂

**功法品质：**
- 凡品（白）- 1.0倍效果
- 灵品（绿）- 1.5倍效果
- 仙品（蓝）- 2.5倍效果
- 神品（紫）- 5.0倍效果
- 圣品（金）- 10.0倍效果

### 三、战斗系统

#### 3.1 战斗流程

```
1. 出剑阶段
   ├── 选择法宝
   ├── 分配剑光（金丹期后）
   ├── 目标选择
   └── 时机判断

2. 破招阶段
   ├── 识破敌方
   ├── 选择对策
   └── 法宝相克

3. 结算阶段
   ├── 伤害计算
   ├── 状态判定
   └── 胜负判定
```

#### 3.2 伤害公式

```python
最终伤害 = (
    基础威力 
    × 境界加成 
    × 五行克制 
    × (1 - 防御减免)
    × 随机波动(0.9-1.1)
)

其中：
- 境界加成 = 1.0 + (攻击方境界 - 防御方境界) × 0.2
- 五行克制 = 1.5（克制）或 0.7（被克）或 1.0（无）
- 防御减免 = 防御力 / (防御力 + 100)
```

#### 3.3 战斗策略

**强攻流**
- 核心：高威力法宝
- 特点：快速结束战斗
- 弱点：消耗大，续航差

**防御流**
- 核心：防御法宝 + 恢复
- 特点：生存能力强
- 弱点：输出低，战斗时间长

**控制流**
- 核心：困敌法宝 + 减益
- 特点：限制敌人行动
- 弱点：需要配合

**爆发流**
- 核心：积蓄真元，一击必杀
- 特点：瞬间高伤害
- 弱点：准备时间长，空窗期弱
