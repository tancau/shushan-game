## 一、世界地图结构

采用**JRPG式区域连接**，通过传送点在各区域间移动：

```
                    【蜀山主峰】
                         |
            +------------+------------+
            |            |            |
      【峨眉山】    【青城山】    【昆仑山】
            |            |            |
      【洞天福地】  【妖魔秘境】  【古战场】
            |            |            |
            +------------+------------+
                         |
                   【终极秘境】
```

## 六、遇敌系统

### 6.1 遇敌机制

**踩地雷式**：
- 在地图上走动随机触发
- 每走一步累积遇敌值
- 达到阈值时触发战斗

### 6.2 遇敌率

| 区域类型 | 遇敌率 | 说明 |
|----------|--------|------|
| 安全区 | 0% | 城镇、门派内部 |
| 低危区 | 5% | 新手区 |
| 中危区 | 10% | 常规区域 |
| 高危区 | 15% | 秘境 |

## 八、传送系统

### 8.1 传送点

- 已探索的城镇
- 已解锁的门派
- 已通过的Boss房间

### 8.2 传送费用

| 距离 | 费用 |
|------|------|
| 同区域 | 免费 |
| 跨区域 | 100灵石 |
| 远距离 | 500灵石 |
