# 《蜀山剑侠传》剧情大纲

> **来源**: 《蜀山剑侠传》还珠楼主

---

## 背景设定

**时间**: 康熙二年（公元1662年）开始

**世界观**: 
- 中国玄幻题材，融合神话、志怪、剑仙、武侠
- 儒释道三家合流的思想体系
- 人能成仙，鸟兽蛇虫皆能得道成妖
- 道行高深者可修炼到元神离开肉体，如遇劫难，九重元神只要逃出一丝一缕，仍可重新修炼

**核心冲突**: 
- 正邪将会进行大决战——"第三次峨眉斗剑"
- 所有的恩怨都会在此了断

---

## 序章：入蜀避祸

**时间**: 康熙二年（公元1662年）

**主要人物**: 李宁、李英琼、周淳、周轻云

**剧情**:
1. 前朝将军李宁带着女儿李英琼入蜀避祸
2. 在巫峡遇到周淳，相约去峨眉后山隐居
3. 周淳之女周轻云被黄山的餐霞道姑收为徒弟
4. 引出剑仙体系，展现蜀山世界的基础设定

**游戏化要点**:
- 新手教程阶段
- 角色创建和基础操作
- 剧情引导玩家进入蜀山世界

## 第一卷：峨眉入门

### 1.1 凝碧崖拜师

**主要人物**: 李英琼、峨眉派众长老

**剧情**:
1. 李英琼因奇遇获得紫郢剑
2. 收服神雕佛奴（钢羽）
3. 收服猩猩袁星
4. 服下朱果，获得紫清神焰兜率火
5. 获得白眉禅师的定珠
6. 拜入峨眉派

**关键事件**:
- 紫郢剑获得（莽苍山）
- 神雕佛奴收服
- 朱果奇遇
- 凝碧崖拜师仪式

**游戏化要点**:
- 主线任务：峨眉入门
- 获得 initial 武器和伙伴
- 学习基础技能

### 1.2 慈云寺之战

**主要人物**: 峨眉派弟子、五台派弟子、智通方丈

**剧情**:
1. 五台派据点慈云寺被正道围攻
2. 峨眉弟子初显身手
3. 正邪初次大规模冲突
4. 绿袍老祖被极乐道人斩做两截

**关键事件**:
- 慈云寺大战
- 绿袍老祖受创

**游戏化要点**:
- 第一个大型副本
- 正邪阵营对抗
- 获得 first major rewards

### 1.3 峨眉开府

**主要人物**: 妙一真人、众仙、各方势力

**剧情**:
1. 峨眉派正式开府，广邀天下群仙
2. 血神子邓隐进犯，被谢山以佛火心灯诛灭
3. 晓月禅师被擒
4. 众弟子正式入门

**关键事件**:
- 峨眉开府大典
- 血神子之战
- 三英二云正式定名

**游戏化要点**:
- 门派系统正式开启
- 高级功法学习
- 主线剧情高潮

## 游戏化建议

### 主线任务线

1. **新手线**: 入蜀避祸 → 峨眉入门 → 慈云寺之战
2. **成长线**: 紫云宫 → 莽苍山 → 幻波池
3. **对抗线**: 绿袍老祖 → 鸠盘婆 → 妖尸谷辰
4. **终极线**: 光明境 → 魔宫大战 → 第三次峨眉斗剑

### 支线任务

- 各门派支线（青城、武当、昆仑等）
- 散仙任务（神驼乙休、枯竹老人等）
- 收集任务（七修剑、各色法宝）
- 伙伴任务（袁星、芝仙等）

### 活动系统

- 峨眉开府（周年庆）
- 正邪大战（PVP活动）
- 天劫（高级挑战）
- 飞升（角色成长里程碑）
