# 第一七九回

带着吉儿以两天两夜赶制出来的精密评估报告，林奇星期一一大早便赶到公司，将资讯研究小组的组长米洛及乔叫进了他的办公室。
他找出第一份评估表和吉儿制作的报表端正地放在他们的面前：“谁能告诉我这是怎么一回事？”
乔和米洛不解地拿起报表浏览一遍。
米洛率先惊呼：“太美了！这是谁做的评估表？用哪一种软体？哪一种机型制作出来的？啧！啧！啧——”
乔满脸不解地抬起头来：“评估结果有很大的差别。”
“当然。问题是哪一份是正确的？”林奇拿出吉儿列印给他的七份不同的报表交给他们，冷冽的眼光逐一审视他们二人：“我要一个非常合理而且能让我满意的解答。”
“这是哪里来的？这不是我们制作的评估表！”米洛莫名其妙地问道。
“你们作的是哪一份？”
“你看到的那一份。”
林奇拿起最初的那份表：“这个？”
他点点头。
“是你亲手作的吗？”
“不是，是共同成果。”
“那其他那些是怎么回事？”
乔忍不住开口：“能不能请你告诉我们，你到底在说什么？”
林奇深吸一口气，提醒自己绝不能莽撞行事，眼前的二个人都是他最信任的左右手：“这七份表全是从公司的中央电脑里叫出来的，而且第一份报表明显地有所偏差！我想弄清楚为什么会有这种情形发生。”
至少第一份报表并未如他和吉儿猜测的已不翼而飞，这使他感觉好过一点，但仍有个谜底尚未解开。
他喜欢开诚布公地谈，宁可弄个清楚，也绝不要在背地里怀疑什么。
米洛这下真的迷糊了：“可是我们只存入一份啊？”
“这就是我要问的。”
乔拿起那些报表，面露怒色：“你怀疑我们之中有人在暗中做了手脚？”
林奇叹口气：“如果我不怀疑，那我一定是个大白痴！但我也信任你们，所以想听听你们的说法。”
乔撇撇唇，脸上再度恢复平静：“我给你的报告是米洛直接交给我的。”
“而我是作出来之后立刻交给乔的。”
“很好。”林奇松了一口气笑道：“那我们现在只要找出在电脑中放入这些资料的是谁就行了。”
米洛不解地搔搔头：“进入电脑公司每个人都能做，可是要进入我们的档案是不可能的，我们上了密码锁啊！除非是自己人，可是知道密码的人没几个。”
“是吗？”林奇叹口气，抽出一堆平面图的密码表：“你是指这个？”
他不可置信地睁大眼睛：“这——不可能——”
乔和林奇心照不宣地交换了一个含笑的眼光：“没什么不可能的，她只花了四个钟头就达成了！”
“天哪——”
“米洛，你负责找出那个制造假评估表的人，另外回去研究两份评估表究竟差异发生在哪里——”他顿了一顿耸耸肩：“还有把保全系统弄好一点可以吗？”
米洛喃喃自语地退了出去，一边念着不可能，另一方面又兴致昂然地研究吉儿破解方程式的独特手法。
林奇站了起来，拍拍乔的肩：“刚刚很抱歉。”
乔不耐地挥挥手：“公事公办，换了我也会这样做。”他望着他，唇角泛起笑容：“吉儿很厉害吧？”
他摇摇头，只能说出二个字：“天才！”
“我一向这么说她！”乔有些得意地笑了起来，随即正色地开口：“我们好像有麻烦了。”
“不是好像，是‘已经’。”林奇在办公椅上坐了下来，皱起眉头：“一份评估表没什么大用处，但对方没理由将一份没用的报表放进电脑等我们发觉，这不合理，其中的内幕很难捉摸。”
“和东立集团脱不了关系。”
“问题是目的到底在于要我们和他们结盟或者破坏结盟？”
二个身经百战的商场悍将面面相觑，一时之间竟也想出合理的解释。
“林先生，里昂先生来拜访。”
“里昂？”他茫然地瞪着话机：“那是谁？”
乔呻吟一声拿出一封信函：“今天才传到我的手上的，他是‘电脑伤害调查委员会’的代表，他要求今天就见你。”
林奇诅咒一声：“叫他进来，另外送三杯咖啡进来。”
门外的秘书答应一声，几乎只等了二秒钟门便被推开，一个黑发男子和一个高挑眩目的金发美女走了进来。
“幸会，林先生。”里昂伸出手。
林奇面带微笑地和他握了手寒暄一番，挑高二道浓眉讯问地看了看那位金发美女。
“这是我的助手希薇儿小姐。”
“您好，林先生。”她甜美地率先伸出手。
“你好。”林奇握了握她出奇柔软的手：“这位是我的秘书乔。”他为他们介绍。
双方客套地招呼，评估的眼光在四个人之间流窜，在确定对方都非泛泛之辈后方坐了下来。秘书小姐送了咖啡进来后退了出去。
里昂微笑地打开公事包：“希望您不介意我们冒昧来访，但我们代表二百家电脑公司，必须恪尽职责。”
“当然。”林奇回答，心里却想着：得罪了他也就公然表示将与全美最有势力的二百家电脑公司为敌，无怪乎这家伙的眼底闪着傲慢。
他和乔心照不宣地对望一眼，乔立刻亲切微笑开口：“我们能为你做些什么？”
希薇儿甜甜地回答：“‘银翼天使恶意事件’您当然知道了吧！我们从资料上知道贵公司是少数幸运没遭到破坏的公司，恐怕必须冒昧请问原因。”
“因为那天全公司的电脑都在保养中，连工厂也不例外，而且我们并非二十四小时运作的公司。”
“但据我们所知，那天并不是周末，为什么挑那天做保养呢？有什么特殊理由吗？”里昂在说这句话之前意味深长地看了林奇一眼，似乎指明他就是事件的主凶。
林奇好脾气地微笑：“并没什么特殊理由，但你们不认为即使那天我并没有下令保养机器，结果也是一样吗？乔刚刚不是说了，我们并非全天候工作的公司，没有人会在凌晨一点还待在公司的。”
“研究小组的人就会。”
“不会。”乔笃定地回答：“我们公司不流行虐待员工，尤其是珍贵的研究员，林奇下令最晚十点以前，他们都必须离开公司，公司的电脑也是设定在晚间十点会自动断电，所以我们不会有人凌晨一点还在工作。”
“这倒是个新鲜的规定。”希薇儿虚伪地笑了起来：“我以为电脑业是门竞争激烈的行业呢！电脑业不是流行分秒必争吗？”
“那是正规的电脑公司，希薇儿小姐似乎忘了林氏企业并非以电脑为主，我们主要的工作在于石油，而不是电脑。”乔淡淡地说道。
“所以伤害当然很轻了。”里昂微笑接下去。
林奇眯起眼睛，看起来已失去他宝贵的耐心：“里昂先生，你在暗示什么吗？”
他笑了起来：“当然没有，我只是说万一的话，贵公司的损失将不会像其他公司一样惨重，有几家公司几乎全毁了，如果再找不到解毒程式可能就要关门大吉了，贵公司当然不会有这种烦恼。”
“这当然是我们的幸运，但也很可能变成我们的不幸不是吗？”乔打量着里昂和希薇儿不怀好意的脸。
“这就很难说了。”希薇儿淡然回答，抽出她的资料夹：“据我所知，贵公司也是中央科学图书馆电脑资料室的使用者之一？”
“是的。”
希薇儿和里昂对望一眼，脸上浮出淡淡笑意：“贵公司当然知道病毒是藉由中央图书馆的电脑资料室传出的吧？”
“是吗？”林奇不经意地挥挥手：“那又如何？使用资料室的只有我们一家公司吗？”
里昂慢吞吞地回答：“当然不是，不过计划要和‘东立’集团合作的公司可就不多了。”
林奇怒不可遏地用力一拍桌子站子起来：“你是来指控我的吗？”
“目前不是。”
林奇怒火更炽：“我可以告诉你，林氏企业和这件事一点关系都没有，如果你硬要将我定罪，请你带着充分的罪证来找我，现在我很忙，没空和你们废话！请出去！”
“你会后悔把我们赶走的！”里昂冷着脸警告。
“你试试看！”他大怒地从他的办公桌后走出。
里昂及希薇儿一接到他那杀人似的目光，忙不迭收拾好东西，仓徨地夺门而出，怨恨的眼神连扫射他们都还没时间！
“你真是暴躁。”乔微笑着抱怨。
“对！我该死的暴躁极了！你枪毙我好了！”他不悦地咕哝。
乔耸耸肩，啜一口冷了的咖啡：“你猜他们来做什么？”
“我瞎了眼才会看不出来，他们当然是来索贿的！”林奇回到自己的位子：“我要是给了，就该死了！”
乔严肃地点点头：“可是你轰走他们只会带来更大的麻烦，懂得利用情势的人都不是傻瓜。”
“叫他们放马过来！”
乔叹口气摇摇头：“你太不懂得商业艺术了！”
他狠狠瞪了他一眼：“那怎么样？给他们一张空白支票？只为了一件根本没做过的事？”
“当然不是，只是——”他哀叹一声，知道说了也没用，林奇的脑袋是高级水泥做的！
“该死的东西！他竟敢这样对待我！”
“别生气，我们会有办法叫他恢复一点理性的！”
“当然！他必须付出一点小小的代价！”
“你打算什么时候打出你的王牌？别等太久，要不然会失效的！”
“不会！那小东西是最后的王牌，我要等到他们全都吃了饵之后，再给他们致命的一击！看看到时候还有谁敢不听话！”
“一年！唉——这个计划可真是费时，你确定一切都没有问题吗？”
“当然！你绝对可以放心，我们很快就会有一大笔一大笔的钱了！”
“你试试看说你自己的名字，然后念这个密码。”吉儿将一切弄好之后说道。
林奇点点头，觉得自己无比笨拙，仿佛今生第一次用自己的名字似的：“林奇，一九五六零四一八。”念完之后，二人都屏息地看着荧幕。
不一会儿，感觉上却像过了一整个世纪似的，荧幕上果然接受了讯息，顺利进入电脑中。
“太棒了！”吉儿大声欢呼，用力拥抱林奇：“我们做到了！”
林奇感染了她的欢欣，将她抱了出来在空中旋转，大笑着凝视她发亮的小脸：“甜心！是你做到了！我什么忙也没帮上！”
她笑着搂住他的脖子，含笑的大眼睛注视着他：“有啊！你提供了你独一无二的声音！”
“你真是个天才！”他大笑用力亲吻她的额：“我真是为你着迷！”
吉儿羞红了脸，嗫嚅着不敢看他。
“怎么啦？小东西，你觉得害羞吗？”他轻笑放下她，点点她的鼻尖：“你比苹果还会脸红！”
她蹲了下来，将脸藏进“约瑟”的长毛里：“我才没有！”她咕哝着。
“是吗？那你为什么不看着我？”他也蹲了下来，抚摸她如丝的短发。
“因为她只是个孤儿！你的‘私人秘书’！”苏西冷冽不悦的声音在门口响起。
“约瑟”低声咆哮着瞪着她，阿林则一脸怒容地站在她的背后：“我试着告诉她，你们正在忙，她却执意要进来！”
吉儿尴尬地站了起来，双手不安地在身前扭绞，苍白的脸直盯着自己的脚尖，身体仍因她的攻击而微微发抖。
“我并没有请你来，苏西。”林奇不悦地说道，双手拉着不断咆哮的“约瑟”。
“什么时候开始我到你这里来已经需要你发邀请函给我了？”她满面怒容尖锐地问道。
他有些罪恶感，这些日子以来，他忙于找出公司的内贼和吉儿替他装的语音控制系统，的确是忽略了苏西——或许是他刻意地不想见到她。
苏西怒气冲冲地转头瞪着阿林：“去给我弄杯茶来！”
阿林黝黑的脸拼命扭曲，他定定地望着林奇，一句话都不说，也没有服从的样子。
“你这——”
“去吧！阿林，顺便麻烦你替吉儿叫辆车上来。”林奇无奈地说道。
“不必了，我自己回去就可以了。”吉儿慌乱地收拾着电脑桌上的东西。
林奇摇摇头：“我不要你一个人走那么黑的路去搭巴士。”
苏西尖锐地哼了一声，迳自在沙发上坐下来，冷眼打量吉儿：“你不认为她太年轻，根本不适合这份工作吗？如果你需要一个懂电脑的私人秘书为什么不找我？我可以替你安排，或者我可以自己替你做。”
他冷冷地看了她一眼，怀疑尊贵的参议员之女懂得使用电脑。“不必！吉儿是最好的，我不需要其他的任何人。”
“是吗？哪一方面是最好的？”她不屑地说道：“一个孤儿？你不认为她——”
“够了！”
吉儿瑟缩一下，雪白的脸显示了她所受的伤害，就在林奇对苏西怒目而视时，她僵硬地移向门口。
“约瑟”凶狠地对着苏西狂吠几声，一脸不善眦牙咧嘴地逼近她。
“滚！离我远一点！”苏西尖叫，拼命用她名贵的小手提包打着“约瑟”的头：“你这只疯狗！我要把你送去野犬收容所，叫他们将你火焚！”
看着她的狼狈样，吉儿羞惭地发觉自己竟有一丝满足，她连忙奔过去拉着“约瑟”，“不要这样‘约瑟’，我们出去。”
在她的安抚下，“约瑟”稍稍平静，它目露凶光地又朝苏西低吼二声，才甘心地和吉儿一起走出门。
苏西气忿地诅咒，连送茶进来的阿林都被她数落得一无是处！阿林不悦地重重放下茶杯转身离去。
“你看看！你看看他们这群没教养的东西！”她尖声大叫面对林奇：“我真不明白你怎么会请这些人来替你工作！不是野蛮的印度人就是没来历的街头流浪儿——”
“住口！”林奇怒吼一跃而起：“你要再敢侮辱他们一个字，我就把你扔出我的房子！”
她狠狠倒抽一口气，不可置信地瞪着他。
他不理会她的惊愕，一个字一个字清晰地说道：“你听清楚，这个房子里的任何一个人——任何一个！都是我最重视的人！如果我胆敢再侮辱他们，我会毫不犹豫地把你那个尊贵的小屁股扔出我家的大门！你听清楚没有！”
苏西半张着口，一脸不信地瞪着他，好半晌才凶猛地站了起来：“你居然为了一个下人、一条笨狗一、一个下贱的女人对我发脾气？你简直是——”
“滚！”
“林奇！”
他阴沉地瞪着她：“你要我扔你出去，还是要自己走？”
她愣了半晌才知道他的威胁是当真的。
苏西——一向目中无人，娇贵的苏西眼眶里含着怨恨、屈辱的泪水哽咽地开口：“林奇！你会后悔的！我要叫爹地取消我们的婚约！你看着好了！我会叫你后悔的！”
“滚！”他狂暴地吼道，暴怒的脸令人感到恐惧！
苏西啜泣着转身冲出门口，不一会儿汽车发动的声音传来，呼啸而去！
“该死！”他怒吼，一手捶在电脑桌上，所有的东西应声震跳一下！
吉儿的小脸和“约瑟”的头悄悄地出现在门口。
她满怀愧疚地开口：“对不起——都是——”
“不关你的事！”他没好气回答。
她瑟缩一下，仿佛被他狠狠打了一巴掌：“那——那我回去了——”
半晌过去，他的心情平静下来：“吉儿——”抬头一看，她已走了，门口再度传来汽车的声音。
林奇再度愤怒地捶了一下桌子！
他不知道自己今晚是着了什么魔了！竟会对苏西这样大吼大叫！然后又迁怒到吉儿的身上。
虽然他一向脾气暴躁，但对女人，他是很温和的，甚至连苏西惯常的抱怨，他也未曾动过怒，但今夜，那些话却叫他怒火中烧。
想到苏西那些话，他叹口气，双手枕到脑后。
他并不特别爱苏西，事实上除了他母亲之外，他从没特别爱过某一个女人，女人对他来说都一样，只有在他有生理需要的时候才会具有意义，连苏西也不例外。
会和她订婚是因为他们双方的家长交情不错，他的父亲和她的父亲提议他们结婚，以便双方的事业，既然他和苏西都没有特别心爱的对像，那么订婚看起来是一件很理所当然的事。
所以他们订婚了，好像商场上签一条互利条约一样，他真的没什么特别的感觉——
直到吉儿出现之前，一切都很正常。没什么好也没什么不好，他向来不相信世界上有爱情这回事，那只是诗人和小说家赖以维生的东西，否则他遇到过那么多的女人，为何不曾想到过爱情！
直到吉儿出现！
仿佛所有的事情都在等待她的出现，她一出现世界便不再正常运作。
每每想起她的一切，他会莫名其妙地微笑，做任何事都不能专心。有一次他竟在开会时微笑了二十分钟，直到所有的人都莫名其妙地瞪着他的傻笑，胆战心惊地看着他反常的行为。如果不是乔推他，他大概会让自己继续出丑下去。
