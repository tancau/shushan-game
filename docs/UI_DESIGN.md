# 界面设计文档

> 蜀山剑侠传 - JRPG界面设计

---

## 一、视觉风格

### 1.1 总体风格：水墨仙侠

- **主色调：** 墨黑 + 宣纸白 + 朱砂红
- **辅色调：** 翡翠绿(HP) + 冰蓝(MP) + 金色(重要)
- **风格：** 水墨渲染、云纹装饰、仙侠意境
- **字体：** 标题用书法体，正文用宋体

### 1.2 色彩系统

| 用途 | 色值 | 说明 |
|------|------|------|
| 背景 | #1a1a2e | 深墨蓝 |
| 宣纸底 | #f5f0e8 | 米白 |
| 主强调 | #c0392b | 朱砂红 |
| HP条 | #27ae60 | 翡翠绿 |
| MP条 | #3498db | 冰蓝 |
| 金色 | #f39c12 | 重要信息 |

## 二、战斗界面

### 2.1 战斗画面布局

```
+--------------------------------------------------+
|  [敌人区域]                                       |
|                                                   |
|     ┌────────┐                                   |
|     │ 妖狼   │  HP: 150/200                      |
|     │ (立绘) │                                   |
|     └────────┘                                   |
|                                                   |
+--------------------------------------------------+
|  [玩家区域]                                       |
|                                                   |
|     ┌────────┐                                   |
|     │ 主角   │  HP: 500/500  MP: 100/100        |
|     │ (立绘) │  Lv.5 练气期                      |
|     └────────┘                                   |
|                                                   |
+--------------------------------------------------+
|  [战斗菜单]                                       |
|  ┌──────┬──────┬──────┬──────┐                  |
|  │ 攻击 │ 技能 │ 物品 │ 防御 │                  |
|  └──────┴──────┴──────┴──────┘                  |

|  主角使用剑斩，造成50点伤害！                      |
+--------------------------------------------------+
```

### 2.2 战斗UI元素

| 元素 | 说明 |
|------|------|
| **HP条** | 绿色渐变，数字显示 |
| **MP条** | 蓝色渐变，数字显示 |
| **行动按钮** | 4个主要按钮 |
| **技能列表** | 弹出式技能菜单 |
| **战斗日志** | 滚动显示战斗信息 |
| **连击显示** | 屏幕右上角 |
