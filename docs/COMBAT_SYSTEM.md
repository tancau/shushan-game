## 一、战斗概述

### 1.1 战斗类型

**经典回合制**：
- 玩家选择行动 → 执行 → 敌人行动 → 循环
- 支持站位系统
- 五行克制机制
- 连击系统

### 1.2 战斗触发

| 触发方式 | 说明 |
|----------|------|
| **踩地雷** | 地图走动随机触发 |
| **剧情战斗** | 剧情事件触发 |
| **Boss战** | 进入Boss区域触发 |

## 三、战斗行动

### 3.1 玩家行动

| 行动 | 说明 | 消耗 |
|------|------|------|
| **攻击** | 普通攻击，无消耗 | 无 |
| **技能** | 使用技能 | MP 10-50 |
| **物品** | 使用丹药/符箓 | 物品数量 |
| **防御** | 减少伤害50%，恢复MP+10 | 无 |
| **逃跑** | 随机成功，Boss战不可逃 | 无 |

### 3.2 行动顺序

```
速度属性决定行动顺序
速度高者先行动
同速度随机决定
```

## 四、五行克制系统

### 4.1 五行关系

```
金 → 木 → 土 → 水 → 火 → 金
```

| 攻击方 | 克制(+50%伤害) | 被克(-40%伤害) |
|--------|----------------|----------------|
| 金 | 木 | 火 |
| 木 | 土 | 金 |
| 水 | 火 | 土 |
| 火 | 金 | 水 |
| 土 | 水 | 木 |

## 七、伤害计算

### 7.1 伤害公式

```
基础伤害 = 攻击力 × 技能倍率

最终伤害 = 基础伤害
           × 五行克制系数
           × 连击加成
           × 暴击系数
           × 随机波动(0.9~1.1)
           × 防御减免
```

### 7.2 防御减免

```
防御减免 = 1 - (敌方防御 / (敌方防御 + 200))
最低伤害 = 基础伤害 × 10%
```
