## 🎨 Figma/Canva 模板建议

### Figma 组件结构
```
📁 Shushan UI Kit
├── 📁 Buttons
│   ├── 📁 Primary (6 components)
│   ├── 📁 Functional (8 components)
│   └── 📁 Status (6 components)
├── 📁 Icons
│   ├── 📁 Status (10 components)
│   ├── 📁 Functional (12 components)
│   ├── 📁 Five Elements (5 components)
│   ├── 📁 Realms (7 components)
│   └── 📁 Operations (16 components)
├── 📁 Panels
│   └── 📁 Backgrounds (10 components)
├── 📁 Progress Bars
│   └── 📁 Variants (15 components)
├── 📁 Color Styles
│   ├── Primary Colors
│   ├── Background Colors
│   └── Status Colors
└── 📁 Text Styles
    ├── Headings (5 styles)
    └── Body Text (3 styles)
```

### Figma 命名规范
- 按钮：`btn-[type]-[name]` (例如: `btn-primary-start-game`)
- 图标：`icon-[category]-[name]` (例如: `icon-status-health`)
- 面板：`panel-[name]` (例如: `panel-character`)
- 进度条：`progress-[name]` (例如: `progress-health`)

### Canva 模板尺寸
- 按钮组：1000×800 px（包含所有按钮）
- 图标集：2000×2000 px（网格排列）
- 面板集：1200×800 px
- 进度条集：800×400 px

## 🖌️ 设计规范

### 边框风格
- **玄金边框**：2-3px 实线 + 内发光
- **圆角**：按钮 8px，面板 12px，图标 4px
- **阴影**：柔和投影，偏移 (0, 4px)，模糊 8px

### 文字风格
- **标题字体**：思源宋体 / Noto Serif SC
- **正文字体**：思源黑体 / Noto Sans SC
- **标题大小**：24px（面板标题）、18px（项目标题）
- **正文大小**：14px（标准）、12px（辅助）

### 动画建议
- 按钮悬停：缩放 1.05 + 发光效果
- 进度条：渐变流动动画
- 图标：轻微呼吸动画（opacity 变化）
