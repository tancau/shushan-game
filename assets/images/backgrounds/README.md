## 🛠️ 推荐生成工具

### 方案 A: Leonardo.ai（推荐）
- **网址**: https://leonardo.ai
- **免费额度**: 每天 150 tokens
- **推荐模型**: 
  - `Leonardo Phoenix`（通用）
  - `Leonardo Anime`（偏卡通风格）
- **优点**: 质量高，支持中文提示词
- **预计消耗**: 每张图约 3-5 tokens

### 方案 B: Bing Image Creator
- **网址**: https://www.bing.com/images/create
- **费用**: 完全免费
- **模型**: DALL-E 3
- **优点**: 免费，质量不错
- **缺点**: 可能需要排队

### 方案 C: Stable Diffusion WebUI（本地）
- **要求**: 需要显卡（推荐 8GB+ VRAM）
- **推荐模型**:
  - `GuoFeng3`（国风模型）
  - `ChilloutMix`（写实风格）
- **优点**: 无限制，完全可控
- **缺点**: 需要硬件支持

## 📋 使用流程

### Leonardo.ai 流程
1. 注册/登录 https://leonardo.ai
2. 选择模型 `Leonardo Phoenix`
3. 复制上述提示词
4. 设置参数:
   - Dimensions: 1920x1080
   - Number of Images: 4（选最好的）
   - Guidance Scale: 7-8
5. 生成后下载最佳图片
6. 重命名为对应文件名

### Bing Image Creator 流程
1. 访问 https://www.bing.com/images/create
2. 粘贴提示词
3. 等待生成（可能需排队）
4. 选择最满意的版本下载

## ✨ 后期优化建议

### 通用优化
1. **色彩调整**: 使用 PS/GIMP 统一色调
2. **UI 适配**: 预留文字显示区域（避免关键内容在边缘）
3. **压缩优化**: 使用 TinyPNG 压缩文件大小

### 风格统一
- 确保所有背景色调一致（建议偏冷色调）
- 统一对比度和饱和度
- 添加统一的滤镜效果

### 备选方案
如果生成效果不理想，可以:
1. 多生成几次选择最佳
2. 使用 Photoshop 合成多个素材
3. 手绘修正细节
4. 聘请画师定制（预算约 50-100 元/张）
