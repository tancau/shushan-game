## 🤖 批量生成建议

### 方案一：AI 绘图工具

| Leonardo.ai | 游戏图标优化 | ⭐⭐⭐⭐ |

### 方案二：批量生成脚本

```bash
# 创建提示词列表文件
cat > prompts.txt << EOF
Chinese Qingzhu Sword, magical flying sword, glowing emerald green energy...
Chinese Ziying Sword, magical flying sword, glowing golden-purple energy...
# ... 更多提示词
EOF

# 使用 API 批量生成 (示例)
while IFS= read -r prompt; do
  curl -X POST "https://api.example.com/generate" \
    -H "Content-Type: application/json" \
    -d "{\"prompt\": \"$prompt\", \"size\": \"64x64\"}"
done < prompts.txt
```
